十进制整数到字符串的转换

今天被问到了这个问题，一下子卡壳，没有回答上来。

十进制比如12345如何转换成字符串？基本思路是要把每一位整数单独提取出来。

一万二千三百四十五，如何提取每一位？其实也简单：

12345 / 10000 = 1; 12345 % 10000 = 2345;

取出的模再来一遍：

2345 / 1000 = 2; 2345 % 1000 = 345;

以此类推。

这算法在我自己的函数库中都有过实现，关键时刻却不记得了。老了吗？sigh~
