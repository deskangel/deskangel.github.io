安装环境

安装 linux 和 linux image 依赖

sudo apt-get build-dep linux linux-image-$(uname -r)

在能成功运行上面的命令之前，可能需要编辑/etc/apt/sources.list，把其中deb-src的行的注释都去掉，然后
